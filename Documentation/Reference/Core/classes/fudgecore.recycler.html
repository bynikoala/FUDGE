Class Recycler

Keeps a depot of objects that have been marked for reuse, sorted by type. Using Recycler reduces load on the carbage collector and thus supports smooth performance

Static borrow

  • borrow<T>(_T: new () => T): T
    • Defined in Documents/HFU/FUDGE/Core/Source/Recycle/Recycler.ts:28

    Returns a reference to an object of the requested type in the depot, but does not remove it there. If no object of the requested type was in the depot, one is created, stored and borrowed. For short term usage of objects in a local scope, when there will be no other call to Recycler.get or .borrow!
