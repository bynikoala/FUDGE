HeightMapFunction

HeightMapFunction: (x: number, z: number) => number
  • Defined in Documents/HFU/FUDGE/Core/Source/Mesh/MeshTerrain.ts:8

This function type takes x and z as Parameters and returns a number - to be used as a heightmap. x and z are mapped from 0 to 1 when used to generate a Heightmap Mesh x * z * 2 represent the amout of faces whiche are created. As a result you get 1 Vertice more in each direction (x and z achsis) For Example: x = 4, z = 4, 16 squares (32 Faces), 25 vertices
